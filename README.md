# AMBA-APB-Protocol
The AMBA APB (Advanced Peripheral Bus) is a low-power, non-pipelined protocol designed for connecting low-bandwidth, low-complexity peripherals in System-on-Chips (SoCs). It relies on a single-master (usually a system bridge) and multi-slave architecture, operating through a three-state Finite State Machine (FSM): IDLE, SETUP, and ACCESS.
